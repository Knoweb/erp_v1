Step 5: Backend Console Instructions

Check the backend console (terminal where Spring Boot is running) for these log messages:

  • When submitting payment: Look for "=== SUBMITTING PAYMENT ===" with company ID and package type
  • After submission: Look for "Payment submitted: receiptId=... companyId=... rowsAffected=1"
  • When fetching pending: Look for "Executing query: SELECT * FROM payment_receipts WHERE status = 'pending'"
  • Payment count: Look for "Found X pending payments"
